Security Vestibule

Planned
About This Project

Construction of a new security vestibule at the High School main entrance, replacing the existing atrium storefront with a new double-door storefront system. The redesigned entry will include a dedicated security desk positioned to manage all incoming visitors and an adjacent new lounge area. This project enhances campus safety by creating a clear and controlled point of entry while maintaining a welcoming environment for the GDS community.
